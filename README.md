## 🚀 About the Project

Hey there! I'm **Devesh Dev Fauzdar**, a creative frontend developer passionate about building immersive, interactive web experiences with modern frameworks and AI-driven solutions. This repository contains the source code for my personal 3D portfolio website.

Designed with **React Three Fiber (R3F)** and **GSAP**, this portfolio breaks the boundaries of traditional web design, offering a cinematic, scroll-driven journey through my experiences, education, and projects.

### ✨ Key Features

- **🎮 Immersive 3D Experience:** A fully scroll-controlled WebGL interactive canvas.
- **🎨 Dynamic Theme Switcher:** Seamless transitions between curated visual themes.
- **📽️ Interactive Project Space:** An animated carousel of top projects.
- **⏳ Timeline Visualizer:** Explore work and education history in a 3D spatial layout.
- **📱 Fully Responsive:** Carefully optimized formatting for both desktop and mobile devices.

## 🧑‍💻 Getting Started Locally

To run this project on your local machine, follow these steps:

### Prerequisites

Ensure you have **Node.js** (v18+) and **npm** installed.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/deveshdevfauzdar/devesh-portfolio-frontend-dev.git
   ```

2. **Navigate into the directory:**
   ```bash
   cd devesh-portfolio-frontend-dev
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

Then, open [http://localhost:3000](http://localhost:3000) in your browser to see the experience!
