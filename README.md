# Devesh Dev Fauzdar — Portfolio

Hey there! I'm Devesh Dev Fauzdar, a creative frontend developer passionate about building immersive, interactive web experiences with modern frameworks and AI-driven solutions.

This is my personal 3D portfolio website. 🚀

## Tech Stack

- Next.js
- React
- TypeScript
- React-three-fiber
- DREI
- GSAP
- Zustand
- Tailwind CSS

## Features

- Immersive 3D scroll-based experience
- Dynamic theme switcher
- Interactive project carousel
- Animated work & education timeline
- Responsive design (desktop & mobile)

## Getting Started

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.
